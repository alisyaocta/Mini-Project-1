**1. ALGORITMA**
1. Mulai Program.
2. Tampilkan menu utama:
   - (1) Melihat Nilai Siswa
   - (2) Menambah Data Siswa
   - (3) Mengubah Nilai Siswa
   - (4) Menghapus Data Siswa
3. Input pilihan menu.
4. Jika memilih menu 1
5. Menampilkan pilihan mata pelajaran.\
   (1) Matematika\
   (2) Bahasa Indonesia\
   (3) Bahasa Inggris
   (1) Input pilihan mapel.
      - Jika pilihan tidak valid, tampilkan pesan error dan berhenti.
      - Jika valid, Tampilkan pilihan metode pencarian:\
        (1) Berdasarkan Nama\
        (2) Berdasarkan Nomor Absen
   (2) Input pilihan metode.
      - Jika memilih metode 1 (Nama).
        - Input nama siswa.
        - Cari indeks siswa berdasarkan nama.
        - Jika ada, tampilkan nilai siswa.
        - Jika tidak ada, tampilkan pesan error.
      - Jika memilih metode 2 (no absen)
        - Input nomor absen siswa.
        - Cari indeks siswa berdasarkan nomor absen.
        - Jika ada, tampilkan nilai siswa.
        - Jika tidak ada, tampilkan pesan error.
6. Jika Memilih Menu 2
   - Input Nama siswa
   - Input Nomor absen
   - Input Nilai Matematika
   - Input Nilai Bahasa Indonesia
   - Nilai Bahasa Inggris
   - Simpan data ke dalam list dan tampilkan data yang berhasil ditambahkan.
7. Jika Memilih Menu 3
   
   (1) Tampilkan daftar mata pelajaran:
     - (1) Matematika
     - (2) Bahasa Indonesia
     - (3) Bahasa Inggris

    (2) Input pilihan mapel.
      - Jika valid, 
      - Jika tidak valid, tampilkan mata pelajaran tidak valid!.
 
    (3) Input nomor absen siswa.
      - Jika nomor absen ada, Input nilai baru.
      - Tampilkan pesan sukses.
      - Jika tidak ada, tampilkan pesan nomor absen tidak valid.

8. Jika Memilih Menu 4
   
   (1) Tampilkan pilihan data yang ingin dihapus:
      - (1) Nama siswa
      - (2) Nomor absen
      - (3) Nilai
   
   (2) Input pilihan hapus.\
   (3) Jika pilih 1, hapus nama siswa dari list berdasarkan  indeks list siswa.\
   (4) Jika pilih 2, hapus nomor absen siswa dari list berdasarkan indeks nomor absen.\
   (5) Jika pilih 3, tampilkan pilihan mapel, lalu hapus nilai siswa dari list berdasarkan indeks mata pelajaran.\
   (6) Jika tidak valid, tampilkan pesan pilihan tidak valid.

    - Tampilkan pesan "Menu tidak valid".
10. Selesai.
